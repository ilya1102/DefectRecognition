# Проект: Распознавание дефектов

## Как запустить:
1. Установить Python
2. Установить библиотеки: `pip install -r requirements.txt`
3. Запустить: `python app.py`

## Тестирование:
В папке проекта лежат 4 примера изображений (img_...). 
Для проверки других изображений просто загрузите их через интерфейс программы.
